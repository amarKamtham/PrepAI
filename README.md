## 📌 Project Overview

PrepAI is an AI-powered interview preparation platform that helps students and job seekers prepare for technical interviews.

The platform allows users to:

- Analyze resumes using AI
- Generate interview questions
- Practice mock interviews
- Receive AI-generated feedback
- Track interview history
- Monitor progress through a personalized dashboard

## 🚀 Installation

### Clone Repository

```bash
git clone https://github.com/amarKamtham/PrepAI.git
```

### Install Backend

```bash
cd backend
npm install
npm start
```

### Install Frontend

```bash
cd frontend
npm install
npm run dev
```
